# 1. Setup development
* Use `node` => v14.20.1 for local development

# 2. Launch in local 
## 2.1. Backend
1. Install dependencies (do frequently on fetching new code): `npm install`
1. Run app by `npm run dev`.

## 2.2 Frontend
1. Install dependencies (do frequently on fetching new code): `npm install`
1. Run app by `npm start`.
